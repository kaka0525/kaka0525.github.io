About the app

A Python educational typing app that allows a single player or multiple players to practice typing randomly generated Python code from GitHub. CopynHaste was built in Django and deployed onto AWS in 4 days. Future plans for CopynHaste include supporting different languages and selecting opponents instead of a random matchup.

The approach

The project was agile based and completed in a 4 days sprint. My team consisted of five engineers and we assigned ourselves to the programming tasks that we were interested in. I was in charge of using the Github API to parse randomly generated code and creating the layout of the application.

The requirements


As an anonymous user, I want to be able to ..


  • Go to registration page and create an account.
  • Look at leader board.

As a registered user, I want to be able to ...


  • Play single player game against computer.
  • To have my name on the leader board if I have the highest score.
  • Not have the option to copy and paste the text in the game.
  • Play asynchronous game against an opponent I choose.
  • Play against a random opponent.
